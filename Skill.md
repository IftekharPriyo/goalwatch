# SKILL.md

## Project Skill Guide

This project is a lightweight Chrome extension for viewing live football scores from the browser toolbar.

## Core Skills Needed

### Frontend

- React
- TypeScript
- Vite
- Component-based UI design
- Basic state management

### Chrome Extension

- Manifest V3
- Popup UI
- Options page
- Chrome Storage API
- Extension permissions
- Loading unpacked extensions for testing

### API Integration

- Fetching data from a football API
- Handling API keys from user input
- Displaying live match data
- Error and loading state handling

### Data Handling

The extension should display:

- Match score
- Current minute or match status
- Goal scorers
- Team names
- Basic match events

## MVP Development Focus

The first version should focus only on:

- Simple popup interface
- User API key setup
- Live match score display
- Goal scorer list

Avoid complex features in the first version, such as authentication, backend, payments, or advanced subscriptions.

## Future Skills

Later versions may include:

- Background refresh
- Goal notifications
- Favorite team tracking
- Backend API proxy
- Caching
- User subscriptions
